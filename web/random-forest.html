Understanding the Basic Architecture

In this deep dive, we'll explore how the decision making of Random Forest is baiscally getting the different decision trees to vote and get the result like it's a election.

Random Forest Class Implementation 
class RandomForest:
    def __init__(self, n_trees=100, min_samples_split=2, max_depth=100, n_feats=None):
        self.n_trees = n_trees
        self.min_samples_split = min_samples_split
        self.max_depth = max_depth
        self.n_feats = n_feats
        self.trees = []

Required Functions:

  • fit - Fit and use the decision tree class to implement models on the data which will be cummulated at alst
  • predict - Make predictions by using all the trained trees

The model starts with all data points at the root and recursively divides the dataset at each step to maximize information gain, creating a tree-like structure of decisions. But just doing it one time we do it again and again for different trees which at last are combined for better predictions.
